Empecemos con la instalacion de Windows 11

Windows 11

1 Agosto de 2021 por Carlos Marcelo Torres Vargas

Todos quieren saber cómo descargar Windows 11 ahora que es oficial y, aunque muchas personas deberían esperar hasta que el nuevo sistema operativo de Microsoft se lance oficialmente en "navidades del 2021", ahora puedes descargar una versión (muy) preliminar para probarlo tú mismo.

Hubo una versión filtrada del nuevo sistema operativo circulando hace un par de semanas, pero de verdad de la buena te recomendamos encarecidamente que no la busques. Lo que vas a descargar de la mayoría de los sitios que dicen tenerlo probablemente va a ser mucho malware, posiblemente incluso bloqueando por completo tu equipo con un clásico ransomware. Ahórrate el estrés y el pago de Bitcoins y descarga Windows 11 de Microsoft solo cuando esté disponible. Después de todo, las navidades no están tan lejos.

Además, la descarga de Windows 11 será una actualización gratuita si tienes Windows 10, por lo que no hay razón para intentar encontrar una versión pirateada de un sistema operativo incompleto. Microsoft ahora también ha lanzado oficialmente una versión preliminar para Windows Insiders, que es una forma mucho más segura de probar Windows 11 antes del lanzamiento oficial. Sin embargo, como se trata de una versión preliminar, es probable que haya errores y funciones incompletas, por lo que solo recomendamos instalar Windows 11 en un PC secundario.

Si solo tienes un PC de confianza para tu día a día, te recomendamos que esperes hasta el lanzamiento oficial de Windows 11. Y ya sabemos que esperar no mola nada, pero cuando se lance, ya sabrás cómo descargar Windows 11 siguiendo esta pequeña guía práctica.

Requisitos Minimos de Windows 11

Para la instalación de windows 11 debemos tener en cuenta los siguientes aspectos:

  • Procesador: 1GHz o más rápido con al menos dos núcleos en un procesador de 64-bits o SoC.
  • RAM: 4GB
  • Almacenamiento: 64GB
  • Firmware del sistema: UEFI, Secure Boot
  • TPM: Trusted Platform Module 2.0
  • Tarjeta gráfica: DirectX 12 o posterior con controlador WDDM 2.0
  • Pantalla: 720p, 8 bits por canal de color, al menos 9 pulgadas en diagonal
  • Conexión a Internet y cuenta de Microsoft: Windows 11 Home requiere una conexión a Internet activa y una cuenta de Microsoft para completar la configuración inicial de primer uso del sistema operativo.

Cómo descargar Windows 11

Antes de empezar, asegúrate de que tu PC cumple con los requisitos mínimos de Windows 11 mencionados anteriormente. Si es así, debes hacer una copia de seguridad de tu PC antes de continuar. Esto garantiza que si algo sale mal (y la actualización a una versión preliminar de un nuevo sistema operativo tiene sus riesgos) tus archivos y configuraciones importantes al menos se podrán restaurar.

Una vez hecho esto, deberás registrarse en el programa Windows Insider.

1. Únete al programa Windows Insider

Para hacer esto, abre la aplicación Configuración y haz clic en 'Actualización y seguridad', luego en 'Programa Windows Insider'.

Es posible que debas activar los datos de diagnóstico opcionales para ver esta opción. Para hacerlo, ve a Configuración> Privacidad> Diagnósticos y comentarios y elige 'Datos de diagnóstico opcionales'.

En la ventana 'Programa Windows Insider', haz clic en 'Comenzar' para unirte. En la ventana que aparece, haz clic en 'Vincular una cuenta', elige tu cuenta de Microsoft y haz clic en 'Continuar'.

2. Únete al canal de desarrollo

Por el momento, la versión de Windows 11 Insider solo está disponible para las personas que usan el canal de desarrollo. Esto está destinado a desarrolladores de aplicaciones y ofrece acceso a versiones muy preliminares de Windows 11. Repetimos, asegúrate de que realmente quieres instalar lo que probablemente sea una compilación bastante anticipada con varios problemas incluidos. Si solo quieres que tu PC funcione normalmente, evita Windows 11 por ahora. Sin embargo, si realmente quieres probar Windows 11, en la página "Elige tu configuración de Insider" que aparece, selecciona "Canal de desarrollo" y luego "Confirmar".

Aparecerá otra ventana con una explicación de lo que puedes esperar de esta versión. Siempre que estés de acuerdo en continuar, haz clic en "Confirmar".

Después tendrás que reiniciar tu PC. Asegúrate de que todo esté guardado y haz clic en 'Reiniciar ahora' en la ventana que aparece.

3. Instala la versión de Windows 11 Insider Preview

Una vez que hayas reiniciado, abre la aplicación Windows Update. Puedes hacer esto buscando 'Windows Update' en la barra de búsqueda de Windows 10. También puedes abrirlo yendo a Configuración> Actualización y seguridad> Actualización de Windows. En la ventana que aparece, haz clic en "Buscar actualizaciones". Debería aparecer la compilación de Windows 11 Insider Preview, y puedes descargarla e instalarla como si fuera una actualización normal de Windows 10.

4. Realiza una instalación limpia de Windows 11

Los pasos anteriores actualizan tu PC de Windows 10 a Windows 11. Sin embargo, si quieres instalar Windows 11 desde cero, puedes descargar la ISO Windows 11 Insider Preview. Asegúrate de estar registrado en el programa Windows Insider y de estar en el canal de desarrollo (que se muestra en los pasos 1 y 2 de esta guía), luego desplázate hasta la parte inferior de la página. En 'Seleccionar edición', elige Windows 11. Una vez que se descargue la ISO, tendrás que crear un USB o DVD de arranque con ella.
